Design and Development

I am a Front-End Developer. I have 1 year commercial experience providing front-end development,producing quality responsive websites and exceptional user experience.

I create responsive websites that allow the user to experience your website in the best and most appropriate way suited to the device they are using. By working using progressive enhancement, your website is delivered with a responsive layout that can best make use of the space available on the smallest to largest devices. Cross browser compatibility is ensured by using feature detection so older browsers still in use today provide a solid experience, whilst modern browsers can go the next step and enhance a user’s time spent on your site.

Rather than create websites on a page-by-page basis, I instead consider each element of your screen-based designs as separate components that can exist elsewhere on a website; outside of the context of your design compositions. With these components I produce a style guide that acts as a document for a reusable and maintainable code base. All involved in a project can rely on a style guide to understand a project’s language and produce a more consistent user experience. My front-end build process involves the use of tools such as Sass, Grunt, Gulp, Bower, and NPM to speed up development.
